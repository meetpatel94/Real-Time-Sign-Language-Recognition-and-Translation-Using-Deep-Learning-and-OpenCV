About GestureAI

Real-Time Hand Gesture Recognition System using Deep Learning and OpenCV

Project Overview

GestureAI is an AI-powered real-time hand gesture recognition system developed as a dissertation project. The system captures hand gestures through a webcam, detects hand landmarks using MediaPipe, classifies gestures using a CNN model, and translates them into readable text.
